# 🛒 Messy Ecommerce Sales Analysis (SQL)

## 📌 Project Overview

This project focuses on analyzing a messy e-commerce sales dataset using SQL.
The dataset contains inconsistencies, missing values, and unstructured fields, making it ideal for practicing data cleaning and transformation techniques.

The goal of this project is to clean the data and extract meaningful business insights.

## 🧹 Data Cleaning Process

The dataset required several preprocessing steps before analysis:

* Handling missing values
* Removing duplicates
* Standardizing text fields (e.g., product names, countries)
* Converting data types (dates, numbers)
* Splitting and formatting columns where needed

---

## 🛠 Tools Used

* SQL Server
* Excel (for initial data inspection)
* Power BI

---

## 📊 Key Business Questions

The following questions were explored during the analysis:

* What is the total revenue over time?
* Who are the top customers by purchase value?
* What are the best-selling products?
* Which countries generate the most sales?
* What are the monthly sales trends?

---

## 📈 Key Insights

* Identified top-performing products and categories
* Found the most valuable customers
* Discovered sales trends across different time periods
* Highlighted data quality issues and how they impact analysis
